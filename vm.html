Installation de la Machine Virtuelle

Nous avons configuré une machine virtuelle Ubuntu en utilisant VirtualBox. Voici les étapes principales :

  • Téléchargement et installation d'Ubuntu depuis le site officiel.
  • Configuration de la langue et du clavier lors de l'installation.
  • Création d'un utilisateur principal.

Pour plus de détails, vous pouvez consulter la vidéo ici.

Déploiement du Site Web

Nous avons procédé à l'installation d'Apache sur la machine virtuelle pour déployer le site web :

  1. Installation des paquets nécessaires : sudo apt install apache2.
  2. Configuration des permissions pour le répertoire web.
  3. Transfert des fichiers HTML/CSS vers le répertoire public /var/www/html.

Le site est accessible à l'adresse IP de la machine virtuelle dans un navigateur.
