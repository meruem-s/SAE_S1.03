Déploiement du Site Web

Pour déployer le site web, voici les étapes que nous avons suivies :

  1. Installation d'Apache pour créer un serveur web :
    2. sudo apt install apache2
  2. Configuration du répertoire racine :
    3. /var/www/html

    Les fichiers HTML et CSS ont été placés dans ce répertoire.

  3. Modification des permissions pour permettre l'accès :
    4. sudo chmod -R 755 /var/www/html
  4. Redémarrage d'Apache pour appliquer les modifications :
    5. sudo systemctl restart apache2
  5. Test du site :

    6. Le site est accessible depuis un navigateur via l'adresse IP de la machine virtuelle.

    http://[IP-de-la-VM]

Résultats

Une fois les étapes terminées, le site s'affiche correctement et permet de naviguer entre les pages HTML/CSS créées.
